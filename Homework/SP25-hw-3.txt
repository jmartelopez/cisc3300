1. Compare and contrast Flexbox and CSS grid, how are they alike and how are they different? 
Flexbox is 1d layout system while CSS grid is 2d layout system. Flexbox deals with either rows or columns. CSS grid deals with rows and columns. Both used for layout systems.
2. What’s responsive design in web development and how do we implement it? 
Responsive design creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it. We implement it using media queries targeting viewport ranges.
